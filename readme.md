# My Portfolio Project

## Overview
This project is a simple personal portfolio website built using **HTML**.  
It includes sections like *About Me*, *Projects*, *Skills*, and *Contact me form*

## Sections Description
1. **Navigation Bar** – Links to About, Projects, and Contact sections.  
2. **About Section** – Contains a welcome message, profile image, and short bio.  
3. **Projects Section** – 3 sample projects with short descriptions.  
4. **Skills Section** – Displays a table of technical skills and proficiency levels.  
5. **Contact Section** – A simple form with Name, Email, and Message fields.  

## Features
- Semantic HTML structure (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)
- Responsive links and form validation
- Clean and readable code for easy CSS styling later

## Validation
All links, form inputs, and HTML structure were checked to ensure correctness and accessibility.

## Clean Code
The project follows clean coding practices with consistent indentation and descriptive labels.
